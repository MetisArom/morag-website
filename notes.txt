HTML NOTES:

basic structure:
<!DOCTYPE html>
<html>
    <head>
        <title>Title</title>
    </head>
    <body>
        content
    </body>
</html>

Sematic tags: See blog.html
<header>: 
<footer>: 
<aside>: 
<main>: 
<article>: 
<nav>: 
<section>: 
<details>: 

tag structure: <tag attributename = "attributevalue"> content </tag>
heading tags: h1 > h2 > h3 > h4 > h5 > h6
break/horizontal lines: <br> and <hr>
text: tag = <p>text</p> <strong> = bold, <em> = italics
links: <a href = "link address url"> link display text</a>
links to external websites: <a href = "url" target = "_blank"> text </a> (opens in new tab)
links/navigation: <nav> <a href = "url 1"> text 1 </a> | <a href = "url 2"> text 2 </a> </nav>
lists:
    Unordered lists: <ul> <li> list item 1 </li> <li> list item 2 </li> </ul>
    Ordered lists: <ol> <li> list item 1 </li> <li> list item 2 </li> </ol>
tables:
<table>
    <thead>
        <tr>
            <th>Head 1</th>
            <th>Head 2</th>
            <th>Head 3</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
            <td>Data 3</td>
        </tr>
    </tbody>
</table>

Forms:
<form action="process.php" method="POST">
    <div>
        <label>First Name</label>
        <input type="text" name="firstName" placeholder="Enter First Name">
    </div>
    <br>
    <div>
        <label>Last Name</label>
        <input type="text" name="lastName" placeholder="Enter Last Name">
    </div>
    <br>
    <div>
        <label>Email</label>
        <input type="email" name="email" placeholder="Enter Email">
    </div>
    <br>
    <div>
        <label>Message</label>
        <textarea name = "message" placeholder="Enter Text"></textarea>
    </div>
    <br>
    <div>
        <label>Gender</label>
        <select name="gender">
            <option value="male">Male</option>
            <option value="female">Female</option>
            <option value="other">Other</option>
        </select>
    </div>
    <br>
    <div>
        <label>Age</label>
        <input type = "number" name = "age" value="18">
    </div>
    <br>
    <div>
        <label>Date</label>
        <input type = "date" name = "date">
    </div>
    <br>
    <div>
        <input type="submit" name="submit" value="Submit">
    </div>
</form> //processed using the file specified in the action attribute

Buttons: <button>Click Me!</button>, add functionality with programming languages/functions

Images <img src="images.jpg" alt="Sample Text" width="x" height="x">, can also wrap in a link

<blockquote>: used for quoting from external sources
<abbr title="Abbreviation Expanation"> Abbr Expla</abbr> = abbreviation explanation as a tooltip
<cite>: used to tell browser that wrapped content is cited from somewhere
